
# MAKEME Project

`makeme` is a build automation tool that uses the README file of an project.

## makeme

Hello World!

### install

We install first all dependencies
```shell
npm install
```

### run

First we need to build the app.
```shell
npm run build:dev
```

Then we run it.
```shell
npm run watch:dev
```
